Monte Carlo Path Tracing

Type: Computer graphics, C++ programming

Description: I implemented a physically-based renderer based on the path tracing algorithm. Path tracing is a simple, elegant Monte Carlo sampling approach to solving the rendering equation. It generates recursive bounce rays in a physically-accurate manner, making it an unbiased estimator for the rendering equation. My path tracer support a wide variety of visual effects (soft shadows, color bleeding, caustics, reflection and refraction, etc.)

Tools: Qt creator, C++
