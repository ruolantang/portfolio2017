Raytracing

Type: Computer graphics, C++ programming

Description: I wrote a fully functioning recursive ray tracer for the cousre Introduction to computer graphics. I shoot out rays from the eye of a camera and calculate intersections with primitives in a scene using their implicit equations. This ray tracer supports reflections, shadows, texture mapping, and some other more complicated rendering techniques.

Tools: Qt creator, C++
